1) first install mongodb on local machine from here https://www.mongodb.com/try/download/community 
2) Run mongodb on port 27017
3) Install npm if you dont have it.
4) Navigate to backend part of project in terminal and run command "npm install" wait for it to install then run "npm run dev"
5) Navigate to frontend part of project in terminal and run command "npm install" wait for it to install then run "npm start"
